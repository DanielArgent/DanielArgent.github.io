theme

Корневой элемент

Имеет название Theme. Содержит слеюдующие аттрибуты:

  • name - имя темы
  • xmlns - пространство имён (должно иметь значение http://danielargent.github.io/xmlspec/lumen-studio/v1/theme)

Константы

В файлах тем доступны константы, которые объвляются в секции Constants. Константы нужны в случаях, когда одно и то же значение указывается в файле темы несколько раз. Переменная объявляется как тег Constant со следующими аттрибутами:

  • name - имя константы
  • type - тип константы
  • value - значение константы
В данной версии доступен лишь тип color. Используются константы путём указания их имени с префиксом # в месте, где подразумевается наличие значения:
<Constants>
    <Constant name="back" type="color" value="2; 44; 55"/>
    <Constant name="fore" type="color" value="108, 149, 79"/>
    <Constant name="lines" type="color" value="7; 54; 66"/>
</Constants>

<Background color="#back"/>
<Foreground color="#fore"/>
<Grid color="#lines"/>
