Метод String.IsDigit?

Показывает, относится ли указанный символ Юникода к категории десятичных цифр. Этот метод определяет, является ли символ строки цифрой десятичной системы счисления. Эта ситуация отличается от применения метода IsNumber?, который определяет является ли символ любым числовым категории Юникода. Номера содержат символы, такие как дроби, индексы, надстрочный, римские цифры, числители валюты, обведенные числа и цифры конкретного скрипта. Допустимые цифры являются членами UnicodeCategory.DecimalDigitNumber категории. Если передаётся строка длины 1, возвращается единственное логическое значение, если длина строки больше 1, возвращается список по одному логическому значению для каждого символа.

puts "6".IsDigit?(); #=> true
puts "A".IsDigit?(); #=> false
puts "6B0!".IsDigit?(); #=> [true, false, true, false]
>>> puts "5".IsDigit?()
True
=> True
>>> puts "A".IsDigit?()
False
=> False
>>> puts "4B5!".IsDigit?()
[True, False, True, False]
=> [True, False, True, False]
