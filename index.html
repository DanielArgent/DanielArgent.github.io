Почему Lumen?

Простота

Lumen очень прост в изучении

Высокоуровневость

Для того, чтобы писать на Lumen не нужно знать тонкости устройства компьютера - он использует абстракции!

Мультипарадигменность

Lumen поддерживает сразу несколько парадигм: процедурное, объектно-ориентированное, обобщённое, функциональное

Интеграция с .NET

Lumen можно легко использовать в своих .NET-приложениях. Он может интегрироваться с любым .NET языком: C#, F#, C++/CLI, PascalABC.NET

Современность

В Lumen реализованы самые современные средства программирования: лямбда-выражения, перегрузка операций, сборка мусора, и многое другое!

Кроссплатформенность

Lumen можно использовать на любой платформе с установленным .NET Framework 4 и выше

Динамика

Lumen позволяет вам писать очень динамичные программы

Большая библиотека

Для Lumen имеется большое количество модулей, написанных на .NET языках, например, gui - модуль для работы с графикой или web - для работы с сетью.

GitHub FAQ

Всё имеет значение

Любая конструкция языка может вернуть какое-либо значение

let a := -9
let b := if a >= 0: a else: -a
print(b) #=> 9

Пользовательские типы данных

Вы запросто можете создать собственный тип данных

[constructor]
let person(name, age) {
    this.name := name
    this.age := age
}
let alex := person("Alex", 18)
print(alex.name) #=> Alex

Подсказки типов

Даже в динамических языках часто приходится беспокоиться о типе данных. Lumen позволяет вам создать подсказки типов, которые будут проверяться во время выполнения.

let sum(x: num, y: num): num := x + y
let res: num := sum(4, 5) # ok
print(sum(5, "h")) #=> TypeError
