Метод String.ToUpper

Переводит символы строки в верхний регистр, и возвращает получившуюся строку. Этот метод не изменяет значение текущего экземпляра. Вместо этого он возвращает новую строку, в которой все символы текущего экземпляра преобразуются в верхний регистр. Также может принимать именованный параметр culture, типа string, содержащий название культуры, согласно которому и будет выполнен перевод.

a = "Hello, world!";
# Переводим строку в верхний регистр и выводим её.
puts a.ToUpper();
>>> a = "Hello, world!"
=> "Hello, world!"
>>> puts a.ToUpper()
hello, world!
=> "hello, world!"

# Переводим все строки списка в верхний регистр.
a = ["first", "second", "trith"];
a.Each!(x => x.ToUpper());
puts a # Выведет ["FIRST", "SECOND", "TRITH"]
>>> a = ["first", "second", "trith"]
=> ["first", "second", "trith"]
>>> a.Each!(x => x.ToUpper())
=> <NULL>
>>> puts a
[FIRST, SECOND, TRITH]
=> ["FIRST", "SECOND", "TRITH"]
