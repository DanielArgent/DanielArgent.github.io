MainForm
Основное окно IDE, содержащее элементы управления и основные методы для работы их визуализции.
Для доступа к главной форме следует использовать статическое свойство Instance этого класса.
Instance создаётся автоматически при запуске приложения, создание допольнительных экземпляров не допускается.
Наследование не допускается.

Основное окно IDE разделено на 4 части:
	TopMenu
	Это основное меню IDE, которое содержит элементы управления самой IDE, проектом, и файловой системой.
	Самый левый элемент этого меню представлен значком среды разработки. Вы не имеете доступа к нему.
	Следующий элемент - это пункт меню TopMenuFile, содержащий основные элементы для управления проектом и 
	файловой системой. Содержит следующие подпункты:
		- TopMenuOpen содержит 2 следующих подпункта:
			- TopMenuOpenFile, предназначенный для открытия файла. Этот элемент недоступен вам.
				Нажатие на этот элемент вызывает открытие диалогового окна выбора файла. В случае, если 
				диалоговое окно возвращает DialogResult.OK - вызывается метод OpenFile статического класса ProjectManager.
			- TopMenuOpenProject, предназначенный для открытия проектов. Этот элемент недоступен вам.
				Нажатие на этот элемент вызывает открытие диалогового окна выбора проекта. В случае, если 
				диалоговое окно возвращает DialogResult.OK - вызывается метод OpenProject статического класса ProjectManager.
		- TopMenuCreate содержит 2 следующих подпункта:
			- TopMenuCreateFile, предназначенный для создания файла. Этот элемент недоступен вам.
				Нажатие на этот элемент вызывает открытие диалогового окна CreateFileDialog выбора файла. 
				В случае, если диалоговое окно возвращает DialogResult.OK - вызывается метод CreateFile статического класса ProjectManager.
			- TopMenuCreateProject, предназначенный для создания проекта. Этот элемент недоступен вам.
				Нажатие на этот элемент вызывает открытие диалогового окна CreateProjectDialog выбора файла. 
				В случае, если диалоговое окно возвращает DialogResult.OK - вызывается метод CreateProject статического класса ProjectManager.
		- TopMenuSave, предназначенный для сохранения текущего открытого файла
			Нажатие на этот элемент вызывает метод SaveFile статического класса ProjectManager. Этот элемент недоступен вам.
			//?:
		- TopMenuSaveAs, предназначенный для сохранения текущего открытого файла в какой либо другой. Этот элемент недоступен вам.
			Нажатие на этот элемент вызывает метод SaveFileAs статического класса ProjectManager.
	Следующий элемент это TopMenuEdit, содержит элементы управления исправлений кода и прочих элементов, содержит следующие
	базовые элементы: 
		- TopMenuEditSeparator. Разделитель между TopMenuUndo и TopMenuRedo, видим только если 
			виден TopMenuUndo или TopMenuRedo. Этот элемент недоступен вам.
		- TopMenuUndo. Отображается только в случае, если ActiveWorkingAreaControl реализует IUndoable.
			Этот элемент недоступен вам.
			Нажатие на этот элемент вызывает вызов метода Undo элемента ActiveWorkingAreaControl
		- TopMenuRedo. Отображается только в случае, если ActiveWorkingAreaControl реализует IRedoable.
			Этот элемент недоступен вам.
			Нажатие на этот элемент вызывает вызов метода Redo элемента ActiveWorkingAreaControl
		- TopMenuCopy. Отображается только в случае, если ActiveWorkingAreaControl реализует ICopyable.
			Этот элемент недоступен вам.
			Нажатие на этот элемент вызывает вызов метода Copy элемента ActiveWorkingAreaControl
		- TopMenuCut. Отображается только в случае, если ActiveWorkingAreaControl реализует ICutable.
			Этот элемент недоступен вам.
			Нажатие на этот элемент вызывает вызов метода Cut элемента ActiveWorkingAreaControl
		- TopMenuPaste. Отображается только в случае, если ActiveWorkingAreaControl реализует IPasteable.
			Этот элемент недоступен вам.
			Нажатие на этот элемент вызывает вызов метода Paste элемента ActiveWorkingAreaControl
		- TopMenuDelete. Отображается только в случае, если ActiveWorkingAreaControl реализует IDeleteable.
			Этот элемент недоступен вам.
			Нажатие на этот элемент вызывает вызов метода Delete элемента ActiveWorkingAreaControl
