Метод String.IsLower?

Определяет, находится ли символ в нижнем регистре. Если передаётся строка длины 1, возвращается единственное логическое значение, если длина строки больше 1, возвращается список по одному логическому значению для каждого символа.

puts "a".IsLower?(); #=> true
puts "A".IsLower?(); #=> false
puts "aBc!".IsLower?(); #=> [true, false, true, false]
>>> puts "a".IsLower?()
True
=> True
>>> puts "A".IsLower?()
False
=> False
>>> puts "aBc!".IsLower?()
[True, False, True, False]
=> [True, False, True, False]
