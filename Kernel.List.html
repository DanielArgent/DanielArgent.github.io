Класс List


Представляет список значений любых типов, доступных по индексу. 
class List: Object, Enumerable, Comparable

Подробное описание

Значения типа Kernel.List - это массивы, автоматически изменяющие свою длину при изменении. Для добавления новых элементов можно воспользоваться методом add! или оператором += . Индексы в коллекции начинаются с нуля. Возможны срезы, а также отрицательные индексы. Тип List является прямым наследником Object. В классе определены методы поиска, сортировки, перечисления, изменения, а также прочие методы для работы со списками. Тип List имеет статическое свойство Empty.

Методы экземпляра

Имя Описание
to_e Преобразует список к типу Kernel.Enumerator.
to_s Преобразует список к типу Kernel.String.
sort Возвращает отсортированную версию списка.
sort! Сортирует список.
compact! Очищает список от элементов, имеющих значение null.
pop! Возвращает и удаляет последний элемент списка.
peek! Возвращает последний элемент списка, но не удаляет его.
shift! Добавляет переданные элементы в начало списка.
unshift! Возвращает первый элемент списка и удаляет его.
each! Модификация метода each. Изменяет список при помощи указанной функции.
add! Добавляет переданные элементы в конец списка.
index Возвращает позицию первого вхождения переданного элемента.
last_index Возвращает позицию последнего вхождения переданного элемента.
clear! Очищает список от всех значений.
exists? Определяет, существует ли элемент, удволетворяющий переданному предикату.
find_index Возвращает индекс первого элемента, удовлетворяющего переданному предикату.
find_last_index Возвращает индекс последнего элемента, удовлетворяющего переданному предикату.
find_last Возвращает первый элемент списка, удовлетворяющий переданному предикату.
insert! Вставляет переданный элемент в указанную позицию списка.
delete! Удаляет элементы списка, равные переданным.
reject! Удаляет элементы списка, удовлетворяющие указанному предикату.
remove_at! Удаляет элемент списка, имеющий переданный индекс.
reverse Возвращает новый список, в котором все элементы списка идут в обратном порядке.
reverse! Реверсирует все элементы списка.
reduce Выполняет редукцию списка переданной функцией.
flatten Разворачивает все вложенные списки.
partition Делит список на два по указанному предикату.
contains? Определяет, содержится ли указанный элемент.
