Комментарии
Различают многострочные и однострочные комментарии. Однострочные начинаются с // и продолжаются до конца строки. Многострочные комментраии располагаются между /* и */. Многострочные комментарии не могут быть вложенными.
Вывод данных.
В Klischee существует два способа вывода данных:
1. Консольный.
Этот способ используется тогда, когда подключена консоль. Имеется два оператора: print и println, которые отличаются тем, что первый, в отличие от второго не переносит курсор на новую строку. Справа от них записывается выражение, которое преобразуется в строку. Например:
print “Hello, world!!!”
>>> Hello, world!!!
println 2*2
>>> 4
2. Оконный.
Этот способ осуществляется при помощи функции alert(). Alert принимает один или два аргумента, первый – сообщение, а второй – заголовок. Например, код alert(“Hello, world!!!”, “Mess”) покажeт окно.
О CKI.
У нас есть интерпретатор - библиотека KlischeeInterpriter, но библиотека не может работать без приложения, поэтому пришлось создать CKI (фактически - консольный интерпретатор клише). После его установки для файлов .ks мы просто делаем "Открыть с помощью" -> "Выбрать программу" -> CKI. Таким образом, теперь двойной клик по любому файлу .ks вызывает CKI. Раньше CKI должен был сначала вызвать консоль(потому и назывался консольным), прочитать текст из файла, который его запустил и отправить его в интерпретатор. Вскоре, однако, вызывать консоль стал сам интерпретатор, и задача CKI свелась лишь к прочтению и отправке содержимого файла. Стоит заметить, что изначально CKI работал только тогда, когда он вызывался двойным кликом по файлу, но если запустить непосредственно exe файл CKI просто на секунду создавал процесс, а затем завершал его, а потому пользователь ничего не видел. Вскоре появилась идея создания в CKI интерпретатора его собственного языка, который помог бы запускать клише-скрипты. Потому сейчас, если зайти в CKI, можно увидеть следующее приглашение:
>>> cki:
Start the command line interpreter CKI...
For information about commands type info
For shutdown enter the command CKI :cki
>>>
Этот интерпретатор читает покомандно и мнгновенно интерпретирует команду. Рассмотрим его основные возможности:
Команды существуют трёх типов:
1. Обычные - имеют синтаксис: команда:
Напимер:
lang: - меняет русский язык на английский и наоборот
cls: - очищает консоль
help: - выводит список всех команд(пока-что нет)
sys: - выводит полные сведения о системе
2. Функциональные - имеют синтаксис: команда[аргумент]:
Это как правило команды, позволяющие управлять файлами и процессами:
del[patch]: - удаляет папку/файл и всё, что в ней находится по указанному пути patch
fi[patch]: - выводит полную информацию о файле
gf[patch]: - получает список файлов в указанной папке
gd[patch]: - получает список папок в указанной папке
md[patch]: - создаёт папку по указанному пути
mf[patch]: - создаёт файл по указанному пути
run[patch]: - запускает скрипт клише по указанному пути
start[name]: - запускает процесс с именем name (может запустить практически любой процесс. если, например, указан сайт, то он откроется в браузере)
read[patch]: - читает содержимое файла и выводит в консоль
3. Режимные - команды, которые перенаправляют вводимые пользователем команды. Существует 3 разновидности:
3.1. Интерпретатор cki - то, где, в принципе, и вводятся эти комманды. Вход в него осуществляется автоматически командой cki:, а выход - :cki. Из него можно попасть в другие режимы.
3.2. Запись в файл начинается с команды fwr[patch]:, заканчивается :fwr. В таком режиме стрелки >>> заменяются на f>>. Всё, что будет записано в этом режиме запишется в файл patch.
3.3. Интерактивный интерпретатор клише начинается с команды iks:, а заканчивается :iks. Всё, что будет введено в этом режиме будет немедленно интерпретированно. В таком режиме стрелки >>> заменяются на i>>.
Кроме того интерпретатор cki поддерживает комментарии, которые начинаются с !!.
Таким образом мы видим, что при помощи консоли cki можно создавать, удалять папки и файлы, записывать в файлы текст, запускать скрипты клише и иные процессы, словом, консоль CKI - полноценная среда разработки.
Ниже приведён пример вычисления факториала от 7 в режиме iks:
>>> cki:
Start the command line interpreter CKI...
For information about commands type info
For shutdown enter the command CKI :cki
>>> iks:
Starting IKS...
i>> def factorial(x) {
i>> result = 1
i>> for i in 1 to x++ result *= i
i>> return result
i>> }
i>> println factorial(7)
5040
i>> :iks
>>> :cki
