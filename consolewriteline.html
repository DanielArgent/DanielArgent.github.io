Функция Console.WriteLine

Преобразовывает переданное значение в строковое представление и выводит результат на консоль, переходя при этом на новую строку.

# Выводит Hello, world! в консоль.
Core.Console.WriteLine("Hello, world!");
>>> Core.Console.WriteLine("Hello, world!")
Hello, world!
=> <NULL>

В случае, если передаётся более одного значения, результат рассматривается как строка форматирования с переданными параметрами.

a = 5.32;
b = true;
Core.Console.WriteLine("a = {0}\nb = {1}", a, b); # Аналогично Core.Console.WriteLine("a = {0}\nb = {1}".Format(a, b))
>>> a = 5.32
=> 5,32
>>> b = true
=> True
>>> Core.Console.WriteLine("a = {0}\nb = {1}", a, b)
a = 5,32
b = True
=> <NULL>
