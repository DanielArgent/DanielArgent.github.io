Функция Console.Read

Получает код очередного символа с консоли.Функция блокируется до ввода входных символов; она завершается при нажатии клавиши Enter. Нажатие Enter добавляет последовательность завершения строки, зависящей от платформы (например, в Windows добавляется последовательность перевод строки + возврат каретки). Последующие вызовы Read метод получения вашего ввод по одному символу за раз. После получения последним символом Read блокирует возврат еще раз и цикл повторяется.

a = Core.Console.Read();
# Напечатает код введенного символа.
puts a;
>>> a = Core.Console.Read()
#
=> 35
>>> puts a
35
=> 35
