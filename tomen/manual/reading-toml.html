Reading TOML

To read TOML file into a .NET object, use Tomen.ReadFile function: 

TomlTable table = Tomen.Tomen.ReadFile("data.toml");

Console.WriteLine(table.Path("person.name.first").AsString()); // "Denis"

To read string with TOML code into a .NET object, use Tomen.ReadString function: 

TomlTable table = Tomen.Tomen.ReadString("x = 9");

Console.WriteLine(table["x"].AsInt()); // 9
