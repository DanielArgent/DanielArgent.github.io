Введение

Эта библиотека позволяет вам работать с языком TOML в ваших .NET приложениях. Вы с лёгкостью можете записывать данные в этот формат, а также читать данные в специальные объекты. Кроме того, реализована полноценная поддержка конвертации между форматами TOML и XML.

Основные типы

Tomen - обеспечивает чтение и запись объектов, а также конвертацию между форматами TOML и XML
ITomlValue - представляет любое значение языка Toml.
TomlTable - представляет таблицу языка Toml.
TomlString - представляет строку языка Toml.
TomlInt - представляет целое число языка Toml.
TomlDouble - представляет число языка Toml.
TomlBool - представляет логическое значение языка Toml.
TomlDate - представляет дату языка Toml.
TomlArray - представляет число языка Toml.

Начало работы

В файле data.toml содержится следующая разметка:
numbers = [1.5, 2.3]

[person]
name = "Alex"
has_hand = true
Мы можем прочитать из него данные при помощи метода ReadFile следующим образом:
TomlTable table = Lumen.Tomen.Tomen.ReadFile("data.toml");

var numbers = table["numbers"] as TomlArray
foreach(var i in numbers)
    Console.WriteLine((i as TomlTable).Value);

var person = table["person"] as TomlTable
Console.WriteLine((table["name"] as TomlString).Value);
Console.WriteLine((table["has_hand"] as TomlBool).Value);
Запись данных в файл обеспечивает метод WriteFile:
TomlTable table = new TomlTable {
    ["x"] = new TomlInt(5),
    ["y"] = new TomlInt(6)
}

Lumen.Tomen.Tomen.WriteFile("output.toml", table);
После выполнения этот код создаст файл output.toml со следующим содержимым:
x = 5
y = 6
