Getting Started

Tomen allows you work easilly with TOML in your .NET applications. You can read your data in TOMl format and read this format into .NET objects back. There are also converter XML/TOML.

Supported features

  • + Comments
  • + Quoted keys
  • + Dotted keys
  • + Basic strings
  • + Multi-line basic strings
  • + Literal strings
  • + Multi-line literal strings
  • + Integers
  • + Floats
  • + Booleans
  • + Offset Date-Time
  • + Local Date-Time
  • + Local Date
  • + Local Time
  • + Arrays
  • + Tables
  • + Inline Tables
  • + Arrays of Tables

Installation

Package Manager: 

Install-Package Tomen -Version 1.0.0

.NET CLI: 

dotnet add package Tomen --version 1.0.0
