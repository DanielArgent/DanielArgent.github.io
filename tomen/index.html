Tomen

use TOML in your .NET applications!

What is it?

Tomen is an open source .NET library that allows you to work easily with TOML files directly in your application.

What it can do?

Tomen allows you to read and write your data in TOML format. Also, there are converter from XML to TOML and vice versa.

What is TOML?

TOML is a widespread human-friendly file format. To learn more about TOML please visit official site.

How can i start?

To start use Tomen just download latest version and read documentation.

How can i help?

You can make bug report or feature request in Tomen GitHub repository.
