Tomen

Парсер TOML для .NET

Введение

Эта библиотека позволяет вам работать с языком TOML в ваших .NET приложениях. Вы с лёгкостью можете записывать данные в этот формат, а также читать данные в специальные объекты. Кроме того, реализована полноценная поддержка конвертации между форматами TOML и XML.

Основные классы

Tomen - главный класс библиотеки, обеспечивающий чтение и запись объектов.
ITomlValue - представляет любое значение языка Toml.
TomlTable - представляет таблицу языка Toml.
TomlString - представляет строку языка Toml.
TomlInt - представляет целое число языка Toml.
TomlDouble - представляет число языка Toml.
TomlBool - представляет логическое значение языка Toml.
TomlDate - представляет дату языка Toml.
TomlArray - представляет число языка Toml.

Поддерживаемые возможности

  •     Комментарии
  •     Ключи в кавычках
  •     Ключи с точкой
  •     Встроенные таблицы
  •     Строки в двойных кавычках
  •     Многострочные строки
  •     Сырые строки
  •     Многострочные сырые строки
  •     Целые числа
  •     Числа с плавающей точкой
  •     Логические значения
  •     Таблицы
  •     Даты и время
  •     Массивы
  •     Массивы таблиц
