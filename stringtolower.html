Метод String.ToLower

Переводит символы строки в нижний регистр, и возвращает получившуюся строку. Этот метод не изменяет значение текущего экземпляра. Вместо этого он возвращает новую строку, в которой все символы текущего экземпляра преобразуются в нижний регистр. Также может принимать именованный параметр culture, типа string, содержащий название культуры, согласно которому и будет выполнен перевод.

a = "Hello, world!";
# Переводим строку в нижний регистр и выводим её.
puts a.ToLower();
>>> a = "Hello, world!"
=> "Hello, world!"
>>> puts a.ToLower()
hello, world!
=> "hello, world!"

# Переводим все строки списка в нижний регистр.
a = ["first", "second", "trith"];
a.Each!(x => x.ToLower());
puts a # Выведет ["first", "second", "trith"]
>>> a = ["FIRST", "SECOND", "TRITH"]
=> ["FIRST", "SECOND", "TRITH"]
>>> a.Each!(x => x.ToLower())
=> <NULL>
>>> puts a
[first, second, trith]
=> ["first", "second", "trith"]
